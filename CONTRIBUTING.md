# Contributing to Awesome AI for Students

Thanks for helping students learn smarter! This list stays useful only because people like you add the tools that genuinely helped them. Contributions of every size are welcome.

## The one rule that matters

> **Does it help a student actually *learn*?**

We add tools that help you understand, research, organize, remember, create, or save time on busywork. We do **not** add tools whose main pitch is "we'll write your essay / do your homework so you don't have to." Learning is the point.

## Quality bar for a new entry

Before you open a PR, make sure your suggestion is:

- **Real and working** — a live, official link (no dead pages, no redirects through ad walls).
- **Relevant to students** — it solves a problem students actually have.
- **Honestly labeled** — mark it accurately with the legend tags.
- **Not a duplicate** — search the README first.
- **Not an affiliate link** — plain official URLs only. This list has zero affiliate links and never will.

## Legend tags

| Tag | Meaning |
|-----|---------|
| 🆓 | Free, or has a genuinely usable free tier |
| 🎓 | Student discount, or free with a student email |
| 🔒 | Privacy-friendly (local, open-source, or doesn't train on your data) |
| ⭐ | Top pick — best-in-class for that category |

## How to add a tool

1. **Fork** this repository.
2. Add your entry to the right category in `README.md`, keeping the format:
   ```markdown
   - [Tool Name](https://official-url.com) — One clear sentence on what it does for a student. 🆓 🎓
   ```
3. Keep the description to **one tight, honest sentence**. No marketing fluff.
4. If you added it to the README, please also add it to `docs/data.js` so it shows up on the visual site (or mention it in your PR and a maintainer will).
5. Open a **Pull Request** with a short note on *why it helped you*.

Not comfortable with Git? Just [open an issue](../../issues/new/choose) suggesting the tool — that's a perfectly good contribution too.

## Removing or fixing entries

Dead link? Tool shut down? Free tier disappeared? Please open an issue or PR — keeping the list accurate is as valuable as adding to it.

## Code of Conduct

By participating, you agree to uphold our [Code of Conduct](CODE_OF_CONDUCT.md). Be kind; we're all here to help students.
